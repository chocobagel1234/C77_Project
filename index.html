How to Download Microsoft Visual Studio

Microsoft Visual Studio is an IDE, or Intergrated Development Environment. This means that all the common developer tools needed to build code for things like applications, webisites and games are all in one graphical user interface, or GUI. To download Visual Studio, you first need to type something like"Microsoft Visual Studio download" on your search engine and then click "Enter". After that, click the first link that should appear, which should say "Download Visual Studio Tools" at the beginning. ( If you're using a search engine that isn't Google, then you may have to scroll down and search a bit more thoroughly for it. ) Then, scroll down until you find both of the Visual Studio options side-by-side or on top of each other. Once you've found it, hover your mouse over the blue drop-down on the 'Visual Studio Code' section and select your version. If you can't find your software version on the main list of options, then click on "More" and the website will take you to another screen where you can try and find the version of software that you are on. You can do this by either clicking on any of the large download buttons or any of the small ones that apply to your specific version. However, if you can find your software version on the main list of options, then click on your version of software and you should be ready to go!

About the Console

The console is a screen which alerts coders - or anyone who can access it (including you!) - of errors and sends messages that have been coded to appear in that particular window. To access it, all you just need to do is right click a blank space on any web page and then, from the menu of options, select "Inspect" and select the "Console" option from the top of the window. This opens the console window, which you can be able to type into - but only if you know what you're doing!

To code a string or interger to appear in the Console window, we can use the console.log( ) function. This "logs" an output (like the value of a variable after it has been changed by another function) into the console. Most programmers prefer to use the Console and the console.log( ) function because it creates a simple user interface in which the user can read and access pieces of information.
